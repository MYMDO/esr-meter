# ESR Meter

[GitHub](https://github.com/MYMDO/esr-meter)

Автоматичний вимірювач внутрішнього опору (ESR) 18650 акумуляторів на базі Arduino Nano, INA226 та ADS1115.

Метод: **DC Pulsed Load** (імпульсне навантаження постійним струмом). Точність: **~ ±0.35%** після калібрування.

## Особливості

- Автоматичне вимірювання ESR раз на секунду
- 4-провідне Kelvin підключення (компенсація contact resistance)
- Гібрид INA226 (струм) + ADS1115 (напруга) — висока точність при мінімумі компонентів
- OLED 0.96" I2C дисплей
- State Machine на millis() — неблокуюча архітектура
- Калібрування через конфігураційний файл
- Візуальна шкала ESR на OLED
- Serial вивід для моніторингу/логінгу
- Захист: визначення підключення батареї, захист від переполюсовки (програмний)

## Як це працює

Внутрішній опір 18650 обчислюється за формулою:

```
ESR = (V_oc - V_load) / I_load
```

1. Вимірюється напруга батареї **без навантаження** (V_oc) — усереднення 16 семплів ADS1115
2. Вмикається MOSFET — підключається резистор R_load = 5.1Ω
3. Через 100ms стабілізації вимірюються **V_load** та **I_load** (16 пар семплів)
4. MOSFET вимикається, обчислюється ESR
5. Результат виводиться на OLED та Serial

**Таймінги:**
- ON: 200ms (100ms стабілізація + ~40ms вимірювання + запас)
- OFF: 800ms (наступний цикл)
- Повний цикл: 1 секунда
- Duty cycle: 20%

### Чому INA226 + ADS1115?

| Параметр | INA226 alone (VBUS) | INA226 + ADS1115 |
|---|---|---|
| LSB для ΔV | 1.25 mV | 0.250 mV (з дільником 2:1) |
| Кроків для ΔV=40mV | 32 | 320 |
| Похибка ESR (50mΩ) | ~±5.5% | ~±0.35% |
| Додаткових компонентів | 0 | 1 модуль + 2 резистори |

### Kelvin 4-провідне підключення

| Клемник | Дріт | Куди | Колір крокодильчика |
|---|---|---|---|
| K1 (Force+) | 22AWG | Drain → (+) батареї | Червоний |
| K2 (Sense+) | 26AWG | Дільник 100kΩ → (+) батареї | Жовтий |
| K3 (Force-) | 22AWG | Source → R_load → INA226 V- → GND | Чорний |
| K4 (Sense-) | 26AWG | ADS1115 GND → (-) батареї | Синій |

**ВАЖЛИВО:** Кожен крокодильчик — окремий контакт на батареї. Не затискати два дроти на один контакт!

## Інструкція збірки

### 1. Підготовка protoboard
Відміряти ~5×7 см, перевірити доріжки мультиметром, зачистити спиртом.

### 2. MOSFET
- **HYG012N03LR1TA (TOLL):** припаяти 3 дротики 26AWG до Gate, Source, Drain. Заізолювати термозбіжкою.
- **IRF3205 (TO-220):** просто вставити в protoboard.

### 3. Встановити колодки
- Arduino Nano (2×15 pin)
- OLED (1×4 pin)
- INA226 (1×4 pin)
- ADS1115 (1×4 pin)
- Клемники (4 шт)

### 4. Розведення
1. **Живлення:** Arduino VIN → USB, 5V → шина (+), GND → шина (-). 100nF + 10µF декаплінг.
2. **I2C:** A4(SDA) та A5(SCL) до всіх модулів. Pull-up 4.7kΩ на 5V.
3. **OLED:** VCC→5V, GND→GND, SDA/SCL→I2C.
4. **INA226:** VCC→5V, GND→GND, SDA/SCL→I2C. V+→GND, V-→R_load. VBUS не підключати.
5. **ADS1115:** VDD→5V, GND→GND, SDA/SCL→I2C, ADDR→GND. A0→дільник.
6. **Дільник:** Sense+ клемник → 100kΩ → A0 ADS1115. A0 → 100kΩ → GND.
7. **Gate:** D2 → 100Ω → Gate MOSFET. Gate → 10kΩ → GND.
8. **Силовий ланцюг:** F+ → Drain. Source → R_load 5.1Ω → INA226 V-. INA226 V+ → GND.

### 5. Перевірка
- Напруга живлення: 5V на всіх модулях
- I2C Scanner: INA226=0x40, ADS1115=0x48, OLED=0x3C
- MOSFET: OFF→ >100kΩ між F+ та F-; ON→ ~5.2Ω (R_load + R_shunt)

## Калібрування

Відредагувати `config.h`:

```c
// Приклад для 2×100kΩ 1%:
#define K_DIVIDER       0.5F     // R2/(R1+R2), виміряти мультиметром

// ESR зміщення (коротке замикання замість батареї):
#define CAL_OFFSET      0.0F     // mOhm - виміряти дроти + контакти

// Мультиплікативна поправка (по референсному резистору):
#define CAL_GAIN        1.0F     // скоригувати після вимірювання відомого R
```

**Порядок:**
1. **Виміряти R_shunt** (між V+ та V- на модулі INA226)
2. **Виміряти дільник:** подати 3.300V на Sense+, виміряти A0 → K_divider = V_A0 / V_sense
3. **ESR offset:** замкнути F+/F- та S+/S- товстим дротом → записати показ → CAL_OFFSET
4. **Gain:** виміряти відомий резистор (напр. 50mΩ) → підігнати CAL_GAIN

## Використання

1. Залити скетч через Arduino IDE або CLI:
```bash
arduino-cli compile --fqbn arduino:avr:nano esr_meter
arduino-cli upload -p /dev/ttyUSB0 --fqbn arduino:avr:nano esr_meter
```

2. Підключити 18650 через 4 крокодильчики (окремі контакти F+/S+ до + та F-/S- до -)
3. На OLED з'явиться ESR в mΩ
4. Serial Monitor (115200 бод) показує детальні дані

**Інтерпретація ESR для 18650:**
- < 30 mΩ: відмінний (premium high-drain)
- 30-75 mΩ: хороший
- 75-150 mΩ: задовільний
- 150-250 mΩ: замінити
- > 250 mΩ: небезпечний (утилізувати)

## Точки налагодження (осцилограф)

| Точка | Сигнал | Норма |
|---|---|---|
| TP1 | D2 Gate drive | 200ms pulse, 0→5V |
| TP2 | Gate MOSFET | τ ≈ 1µs через 100Ω |
| TP3 | Drain → F+ | V_oc ≈ 3.7V, V_load ≈ 3.66V |
| TP4 | INA226 V- | ~71mV при 0.71A |
| TP5 | ADS1115 A0 | 1.85-2.1V (через дільник) |
| TP6 | Sense+ / Sense- | Те ж, що Drain/GND |

## Специфікація MOSFET

| MOSFET | Корпус | Rds @5V | VGS(th) | Монтаж |
|---|---|---|---|---|
| **HYG012N03LR1TA** | TOLL SMD | 1.0mΩ typ | 1-3V | Пайка дротів |
| MDU04N010VRH | PDFN56 SMD | ~1.5mΩ | 1-2V | Пайка дротів |
| IRF3205 | TO-220 | ~15mΩ | 2-4V | Protoboard |

RDS(on) MOSFET **не впливає** на точність вимірювання (струм вимірюється безпосередньо INA226).

## Точність

| Компонент | Похибка |
|---|---|
| INA226 current (gain + offset) | ±0.10% |
| ADS1115 ΔV (noise + INL + quant.) | ±0.52% |
| **ESR total (RSS)** | **±0.53%** |
| Після калібрування | **~±0.35%** |

Для 50mΩ cell: ±0.35% = ±0.175 mΩ.

## Troubleshooting

| Проблема | Причина | Рішення |
|---|---|---|
| OLED порожній | I2C адреса не 0x3C | I2C Scanner |
| INA226 не знайдено | A0/A1 не на GND | Перевірити адресацію |
| V_oc = 0V | Батарея не підключена | Перевірити крокодильчики |
| I_load = 0 при ON | MOSFET не відкривається | D2→100Ω→Gate; перевірити 10kΩ pulldown |
| I_load > 0 при OFF | MOSFET не закривається | 10kΩ pulldown відсутній |
| ESR від'ємний | Kelvin неправильно | Переплутані Sense+/Sense- |
| ESR нестабільний | Contact resistance | Зачистити контакти |
| R_load гарячий | Duty cycle > 20% | ON ≤200ms, OFF ≥800ms |
